# 🧩 Job-Resume Matching System
**Semantic Similarity Based Framework for Candidate–Role Alignment**  
*Tech Stack:* Python, LangChain, BERT, FAISS, Flask  

---

### 🔍 Overview  
An NLP-driven application that aligns resumes with job descriptions using **BERT embeddings** and **FAISS vector search**, enhanced with **LangChain reasoning** to improve matching accuracy and explainability.  

---

### ⚙️ Key Features  
- **Semantic Matching:** Computes contextual similarity between resumes and JDs using BERT and FAISS.  
- **LangChain Reasoning:** Adds interpretability by explaining matching logic in natural language.  
- **Automated Skill Extraction:** Parses and ranks skills via REST APIs for recruiter integration.  
- **Prompt Validation:** Fine-tunes alignment with feedback-based prompts.  

---

### 📈 Results  
- Improved candidate–role alignment through contextual semantic matching.  
- Reduced manual screening effort via automated resume ranking and skill extraction.  

---

### 🛠️ Future Enhancements  
- Integrate GPT-based summarization for richer candidate insights.
